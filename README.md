# 📊 AI Customer Insights Dashboard

An AI-powered dashboard that analyzes customer complaints and generates structured business insights.

## 🚀 Features
- Analyze customer complaints using AI
- Classify issues by severity
- Visualize insights with charts
- Upload files or paste text input

## 🛠 Tech Stack
- Python
- Streamlit
- OpenAI API
- Pandas

## ▶️ Run Locally
```bash
streamlit run app/app.py
